## 📞 Support & Troubleshooting

### Common Issues

**Issue:** Charts not rendering
- **Solution:** Check matplotlib/seaborn installed
- **Fallback:** System shows placeholder + text

**Issue:** LLM analysis slow
- **Solution:** Check Ollama running on localhost:11434
- **Fallback:** Uses comprehensive default data

**Issue:** Forecast accuracy low (R² < 0.6)
- **Solution:** Check data quality and consistency
- **Display:** System warns user of confidence level

**Issue:** API connection error
- **Solution:** Verify backend running on port 5000
- **Check:** firewall/CORS settings

## 🚀 Deployment Instructions

### Local Development
```bash
# Backend
cd backend
pip install -r requirements.txt
python run.py  # Runs on localhost:5000

# Frontend  
cd frontend
npm install
npm run dev  # Runs on localhost:3000

# Ollama (in separate terminal)
ollama serve  # Default port 11434
ollama pull deepseek-r1:14b  # Download model
```

### Production (Docker)
```bash
docker-compose up --build
# Compose includes:
# - Frontend (port 3000)
# - Backend (port 5000)
# - Ollama (port 11434)
# - MongoDB (optional)
```
